xmldoc

Syntax
xmldoc --help
xmldoc [-ilf] workdir cfgname xmldoc [search]
Synopsis

XML documentation system document parser.

Options
--helpprint the help message and exit.
-ilfignore line feeds in the XML document.
workdirworking directory to operate in.
cfgnamefilename of the configuration file.
xmldocfilename of the XML document to process.
searchsearch text.
