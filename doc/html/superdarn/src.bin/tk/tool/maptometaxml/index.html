maptometaxml

Syntax
maptometaxml --help
maptometaxml [-new] name
Synopsis

Extract meta-data from a map format file and store them as an XML document.

Options
--helpprint the help message and exit.
-newthe input file is a cnvmap format file.
namefilename of the map format file to trim. If this is omitted, the file is read from standard input.
Description

Extract meta-data from a map format file and store them as an XML document.

The XML is written to standard output.

Examples
maptometaxml 20040830.map > 20040830.map.xml

Extract meta-data from the file "20040830.map" and write it to the file "20040830.map.xml".
