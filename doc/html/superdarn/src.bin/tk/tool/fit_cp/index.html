fit_cp

Syntax
fit_cp --help
fit_cp fitnames...
fit_cp -new [fitacfname]
Synopsis

Determines the program identifier numbers in fit or fitacf format files.

Options
--helpprint the help message and exit.
fitnamesfilenames of the fit format files.
-newfiles are in the fitacf format.
fitacfnamefilename of the fitacf format file. If this is omitted the file is read from standard input.
Description

Determines the program identifier numbers in fit or fitacf format files.

The total number of program identifiers followed by a list of the identifier numbers is written to standard output.

Examples
fit_cp 20030408.gbr.fit

Find the program identifier numbers in the fit file "20030408.gbr.fit".
