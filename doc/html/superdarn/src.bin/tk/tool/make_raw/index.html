make_raw

Syntax
make_raw --help
make_raw [-vb] [iqdatname]
Synopsis

Creates a rawacf format file from a iqdat format file.

Options
--helpprint the help message and exit.
-vbverbose. Log information to the console.
iqdatnamefilename of the iqdat format file. If this is omitted the file is read from standard input.
Description

Creates a rawacf format file from a iqdat format file.

The rawacf format file is written to standard output.

Examples
make_raw -new 20040830.gbr.iqdat > 20040830.gbr.rawacf

Create an rawacf format file from the iqdat format file "20040830.gbr.iqdat".The rawacf format ile is written to "20040830.gbr.rawacf".
