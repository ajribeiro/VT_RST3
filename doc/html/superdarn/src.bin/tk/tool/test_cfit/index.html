test_cfit

Syntax
test_cfit --help
test_cfit names...
Synopsis

Tests the integrity of cfit format files by reading them and printing the contents.

Options
--helpprint the help message and exit.
namesfilenames of the vfit format files.
Description

Tests the integrity of cfit format files by reading them and printing the contents.

The contents are written to standard output

Examples
test_cfit 20040830.gbr.cfit | more

Test the integrity of the file "20040830.gbr.cfit", paginating using more.
