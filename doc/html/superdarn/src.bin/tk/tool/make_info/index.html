make_info

Syntax
make_info --help
make_info fitnames...
make_info -new [fitacfname]
make_info -cfit cfitnames...
Synopsis

Extract information (metadata) from fit, fitacf or cfit format files.

Options
--helpprint the help message and exit.
fitnamesfilenames of the fit format files.
-newfiles are in the fitacf format.
fitacfnamefilename of the fitacf format file. If this is omitted the file is read from standard input.
-cfitfiles are in the cfit format.
cfitnamesfilenames of the cfit format files.
Description

Extract information (metadata) from fit, fitacf or cfit format files.

The extracted information is written to standard output.

Examples
make_info 20040830.gbr.fit > 20040830.gbr.info

Extract information from the fit file "20040830.gbr.fit" and store it in the file "20040830.gbr.info". 

make_info -new 20040830.gbr.fitacf > 20040830.gbr.info

Extract information from the fitacf file "20040830.gbr.fitacf" and store it in the file "20040830.gbr.info".
