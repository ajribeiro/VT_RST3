test_raw

Syntax
test_raw --help
test_raw rawnames...
test_raw -new [rawacfname]
Synopsis

Tests the integrity of raw (dat) or rawacf format files by reading them and printing the contents.

Options
--helpprint the help message and exit.
rawnamesfilenames of the raw (dat) format files.
-newfiles are in the rawacf format.
rawacfnamefilename of the rawacf format file. If this is omitted the file is read from standard input.
Description

Tests the integrity of raw (dat) or rawacf format files by reading them and printing the contents.

The contents are written to standard output

Examples
test_raw 20040830.gbr.dat | more

Test the integrity of the file "20040830.gbr.dat", paginating using more.
