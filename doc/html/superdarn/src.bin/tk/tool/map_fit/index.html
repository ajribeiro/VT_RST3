map_fit

Syntax
map_fit --help
map_fit [-vb] [-new] [-ew errwgt] [-mw modelwgt] [-s source] [mapname]
Synopsis
None
Options
--helpprint the help message and exit.
-vbverbose. Log information to the console.
-newthe input file is a cnvmap format file.
-ew errwgtset the error weighting to errwgt, which can be either yes or no.
-mw modelwgtset the model weighting to errwgt, which can be either yes or no.
-s sourceset the text string indicating the data source to source.
mapnamefilename of the map format file. If this is omitted, the file will be read from standard input.
Description
None
Examples
