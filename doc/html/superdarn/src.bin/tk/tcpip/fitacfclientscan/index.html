fitacfclientscan

Syntax
fitacfclientscan --help
fitacfclientscan [-s stream] host port
Synopsis

Simple client program for reading complete scans from a fitacf TCP/IP data stream.

Options
--helpprint the help message and exit.
-s streamfor a multiplexed stream, read the stream number given by stream. A value of 255 will read all streams
hosthostname or IP address of the system to connect to.
portport number to connect to on the system.
Description

Simple client program for reading complete scans from a fitacf TCP/IP data stream.

This task is for demonstration purposes.

Examples
fitacfclientscan peanut.jhuapl.edu 1024

Connect to the host peanut.jhuapl.edu at port 1024 and download complete scans.

fitacfclientscan -s 2 peanut.jhuapl.edu 1024

Connect to the host peanut.jhuapl.edu at port 1024 and download complete scans from stream number 2.
