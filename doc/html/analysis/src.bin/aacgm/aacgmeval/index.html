aacgmeval

Syntax
aacgmeval --help
aacgmeval [-i] [-lat lat] [-lon lon] [-alt alt] [-fmt format]
aacgmeval [-i] [-f filename] [-fmt format]
Synopsis

Convert to and from Altitude Adjusted Corrected Geo-Magnetic Coordinates (AACGM).

Options
--helpprint the help message and exit.
-iperform the inverse conversion from AACGM to geographic.
-lat latcalculate for the latitude lat.
-lon loncalculate for the longitude lon.
-alt mloncalculate for the altitude in kilometers, e alt.
-fmt formatuse the string fmt, to format the output.
-f filenameread a sequence of dates, times and longitudes from the ASCII file, filename.
Description

Converts to and from Altitude Adjusted Corrected Geo-Magnetic Coordinates (AACGM).The output is written to standard output using the formatting string given by the "-fmt" option. The format string should be in the same format as that used by the printf function and whill take two double precision floating point numbers as arguments.

If the "-f" option is given, the latitude,longitude and altitude are read from a space separated ASCII file. If the filename is "-" input is read from standard input.

Examples
aacgmeval -lat 85.4 -lon 32.4 -alt 300.0

Convert the coordinate given by latitude of 85.4 degrees, longitude 32.4 degrees and altitude of 300 kilometes to AACGM.

aacgmeval  -i -f aacgm.txt

Convert the AACGM coordinated in the file "aacgm.txt", back to geographic coordinates.
